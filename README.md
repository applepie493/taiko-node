# taiko-node

## 1.リポリトジの複製
```
git clone https://github.com/eth-educators/eth-docker
cd eth-docker
```

## 2.アーカイブノードを有効にする
```
cp default.env .env
```
.envファイルを編集する。
```
nano .env
```
